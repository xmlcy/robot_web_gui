# robot_web_gui

#### 一个很小的 robot web gui 框架

web界面使用纯静态页面实现，后台使用 python 基于 websockets 实现

### front end

web界面使用纯静态页面实现 HTML CSS JS

### back end

使用 python 基于 websockets 实现

``` shell
pip install websockets
python server.py
python clinet.py
```

### 使用

1. 开启服务端

``` shell
python server.py
```

2. 使用浏览器打开 front_end/index.html

3. 选择表情演示 点击表情菜单中的按钮即可在服务端看到消息
